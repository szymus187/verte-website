# Verte Website Improvements Design

## Context
Verte is a Polish B2B marketing website for document automation services targeting SMBs in the Silesia region. The site is well-designed but needs comprehensive improvements in copy, SEO, accessibility, and conversion optimization.

## Skills Applied
- **copywriting** - Rewrite marketing copy for better conversion
- **frontend-design** - UI component improvements
- **marketing-ideas** - Conversion optimization elements
- **tailwind-design-system** - Design token standardization
- Manual: SEO technical optimization, accessibility compliance, performance

## Improvements

### 1. SEO & Meta
- Open Graph + Twitter Card meta tags
- Schema.org JSON-LD (Organization, LocalBusiness, Service, FAQ)
- Canonical URL
- Improved meta description

### 2. Hero Copy & Design
- Problem-first headline with specific number (13h/week)
- Specific CTA: "Umow 15-min rozmowe"
- Social proof under CTAs

### 3. Service Cards Copy
- Benefit-focused titles instead of feature descriptions
- Pain-first, then solution approach
- Micro-stats per card

### 4. Case Study Enhancement
- Stronger testimonial context
- CTA after case study
- Better visual hierarchy

### 5. KSeF Urgency
- Countdown/timeline urgency
- Free audit CTA
- Expanded FAQ

### 6. About Section
- Consistent "I" voice (founder-led)
- Expanded storytelling
- Tech badges reframed as client benefits

### 7. Contact Conversion
- Response time promise (24h)
- Better form validation with feedback
- Calendly-style alternative CTA

### 8. Footer Trust Signals
- Privacy policy link
- Social media placeholders
- Trust indicators

### 9. Accessibility & Performance
- WCAG AA contrast fixes
- Skip-to-content link
- Focus styles improvement
- Lazy loading
- Font preloading

## Approach
- Full HTML/CSS/JS freedom (approved by user)
- Use all applicable MCP skills
- Polish language throughout
